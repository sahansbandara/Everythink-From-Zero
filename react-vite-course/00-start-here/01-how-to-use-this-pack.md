# How to Use This Pack

## Goal
This pack is meant to give you a **clean learning path** through React + Vite.

## What to do
1. Read each section in order.
2. Do the quiz at the end of each major stage.
3. Write down concepts in your own words.
4. Build the mini projects.
5. Review the cheat sheets repeatedly.

## Learning rule
If you cannot explain a term in simple words, you do not understand it yet.

## Good study method
For each concept:
- definition
- why it exists
- where it fits
- one example
- one common mistake

## Bad study method
- watching tutorials passively
- copying code with no understanding
- memorizing commands without knowing what they do
- skipping JavaScript fundamentals

## Skill check
At the end of each folder, ask:
- What problem does this solve?
- Where is this used?
- What breaks if I misuse it?
