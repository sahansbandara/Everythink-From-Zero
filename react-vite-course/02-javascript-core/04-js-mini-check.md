# JavaScript Mini Check

## Questions
1. What is the difference between `let` and `const`?
2. What does `.map()` do?
3. What is destructuring?
4. What does `async` mean?
5. Why do React components often use arrays and objects?

## Sample answers
1. `const` cannot be reassigned; `let` can.
2. It creates a new array by transforming each item.
3. It pulls values out of objects/arrays into variables.
4. It marks a function that can use `await`.
5. Because UI data often comes as lists and structured objects.
