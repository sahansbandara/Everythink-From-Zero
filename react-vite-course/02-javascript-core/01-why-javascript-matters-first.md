# Why JavaScript Matters Before React

## Hard truth
If your JavaScript is weak, your React will also be weak.

React is not magic.
It is mostly:
- JavaScript
- component structure
- React-specific APIs

## What you must know before React
- variables
- functions
- arrays
- objects
- conditions
- loops
- events
- modules
- async/await
- fetch
- basic DOM concepts

## Why beginners fail
They memorize:
- `useState`
- `props`
- `npm run dev`

But they cannot explain:
- array `.map()`
- object destructuring
- arrow functions
- async code

That creates shallow skill.
