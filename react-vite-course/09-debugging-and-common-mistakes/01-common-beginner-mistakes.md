# Common Beginner Mistakes

1. Thinking React is a language
2. Thinking Vite is the same thing as React
3. Skipping JavaScript basics
4. Pushing `node_modules`
5. Storing secrets in frontend code
6. Writing giant components
7. Confusing props and state
8. Forgetting `key` in lists
9. Breaking controlled inputs
10. Causing infinite `useEffect` loops
11. Not handling loading/error states
12. Using `<a>` instead of router links for app navigation
13. Overusing Context when local state is enough
14. Using optimization hooks without need
15. Copy-pasting code with no understanding

## Brutal truth
Most beginner bugs are not “advanced React problems.”
They are basic mental model problems.
