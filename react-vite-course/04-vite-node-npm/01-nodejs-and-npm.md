# Node.js and npm

## Node.js
Node.js lets JavaScript run outside the browser.

Why it matters:
- Vite uses it
- npm uses it
- build scripts use it

## npm
npm is the package manager.

It does 2 big things:
1. installs packages
2. runs scripts

## Common commands
```bash
npm install
npm run dev
npm run build
```

## What they mean
### `npm install`
Read `package.json` and install dependencies.

### `npm run dev`
Run the `dev` script from `package.json`.

### `npm run build`
Run the `build` script from `package.json`.
