# Create, Run, and Build a React + Vite Project

## Command flow
```bash
npm create vite@latest my-app
cd my-app
npm install
npm run dev
```

## Build for production
```bash
npm run build
```

## What build creates
Usually a `dist/` folder.

## Development vs production

### Development
- local server
- faster iteration
- easier debugging

### Production
- optimized files
- smaller bundles
- deployment ready
