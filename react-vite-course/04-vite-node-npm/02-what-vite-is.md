# What Vite Is

## Correct definition
Vite is a frontend build tool and development server.

## What it helps with
- fast dev server
- hot reload / fast refresh
- bundling
- optimized production build

## Simple comparison
React is for writing the UI.
Vite is for serving/building the project.

## Why Vite became popular
Older toolchains often felt slower and more painful.
Vite improved developer experience by being simpler and faster.
