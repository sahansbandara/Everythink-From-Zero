# Command Reference

## Create project
```bash
npm create vite@latest my-app
```

## Enter project
```bash
cd my-app
```

## Install dependencies
```bash
npm install
```

## Start development server
```bash
npm run dev
```

## Build for production
```bash
npm run build
```

## See generated production output
Usually:
```text
dist/
```

## Install a package
```bash
npm install axios
```

## Install a dev dependency
```bash
npm install -D eslint
```
