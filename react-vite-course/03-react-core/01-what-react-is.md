# What React Is

## Correct definition
React is a **JavaScript library for building user interfaces**.

## Not these
React is not:
- a language
- a database
- a server
- a CSS framework

## Main idea
React helps you build complex UIs using reusable pieces called components.

## Why React exists
Without a good UI system, large apps become:
- repetitive
- hard to update
- hard to debug
- hard to scale

React gives structure.

## Visual model

```text
Big app
 ↓
Split into small parts
 ↓
Components
 ↓
Reusable and easier to manage
```
