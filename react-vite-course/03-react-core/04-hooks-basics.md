# Hooks Basics

## What is a hook?
A hook is a special React function that lets function components use React features.

## Common hooks
- `useState`
- `useEffect`
- `useRef`
- `useMemo`
- `useCallback`
- `useContext`

## Most important beginner hooks
### `useState`
Stores changing data.

### `useEffect`
Runs side effects like:
- API calls
- timers
- subscriptions
- localStorage sync

## Why hooks exist
Before hooks, class components handled a lot of this logic.
Hooks made function components much more powerful and cleaner.
