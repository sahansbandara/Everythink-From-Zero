# HTML, CSS, and JavaScript

## HTML
HTML defines structure.

Example:
```html
<h1>Hello</h1>
<p>This is a paragraph.</p>
<button>Click</button>
```

Think of HTML as the **skeleton**.

## CSS
CSS defines appearance.

Example:
```css
button {
  background: blue;
  color: white;
}
```

Think of CSS as the **skin and clothing**.

## JavaScript
JavaScript controls behavior.

Example:
```javascript
console.log("Button clicked");
```

Think of JavaScript as the **brain and behavior**.

## Visual model

```text
HTML  = structure
CSS   = style
JS    = behavior
```

## Common confusion
React does not replace JavaScript.
React is built with JavaScript.
