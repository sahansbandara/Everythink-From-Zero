# What the Web Is

## Simple model

```text
User opens browser
        ↓
Browser requests files
        ↓
Server sends HTML/CSS/JS/assets
        ↓
Browser renders page
```

## Basic parts of the web
- browser
- server
- internet connection
- files/resources
- user interactions

## Website vs web app

### Website
Usually informational.
Examples:
- company page
- blog
- portfolio

### Web app
More interactive and logic-heavy.
Examples:
- dashboard
- booking system
- chat app
- admin panel
- finance portal

## Why this matters
React is mostly useful when the UI becomes dynamic and complex.

## Core reality
The browser directly understands:
- HTML
- CSS
- JavaScript

Everything else is built around those.
