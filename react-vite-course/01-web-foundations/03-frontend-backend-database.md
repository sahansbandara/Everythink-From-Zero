# Frontend vs Backend vs Database

## Frontend
What the user sees and interacts with.

Examples:
- buttons
- inputs
- cards
- pages
- tables
- navigation

## Backend
Runs on the server.

Examples:
- auth logic
- payment logic
- validation
- database queries
- sending emails

## Database
Stores information.

Examples:
- users
- products
- messages
- appointments
- orders

## Visual model

```text
User
 ↓
Frontend (React app in browser)
 ↓
Backend (server/API)
 ↓
Database
```

## Important
React is mainly frontend.
It does not replace the backend.
