# Flashcards

## Card 1
**Q:** What is React?  
**A:** A JavaScript library for building user interfaces.

## Card 2
**Q:** What is Vite?  
**A:** A fast development server and build tool for frontend projects.

## Card 3
**Q:** What is JSX?  
**A:** HTML-like syntax used in React code.

## Card 4
**Q:** Props vs state?  
**A:** Props come into a component; state lives inside a component and changes.

## Card 5
**Q:** Why avoid pushing `node_modules`?  
**A:** It is large, generated, and can be recreated with `npm install`.

## Card 6
**Q:** What does `npm run build` do?  
**A:** Creates the optimized production output.

## Card 7
**Q:** What does `useEffect` do?  
**A:** Runs side effects such as API calls or syncing to localStorage.

## Card 8
**Q:** What is controlled input?  
**A:** An input whose value is controlled by React state.

## Card 9
**Q:** What is routing?  
**A:** Showing different views/pages based on the URL.

## Card 10
**Q:** What is a custom hook?  
**A:** A reusable function containing React hook logic.
