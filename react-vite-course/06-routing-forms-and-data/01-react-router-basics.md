# React Router Basics

## What routing is
Routing lets your app show different pages/views.

Examples:
- `/`
- `/about`
- `/dashboard`
- `/profile`

## Common package
`react-router-dom`

## Why routing matters
Most real apps are multi-page in behavior, even if they are single-page apps underneath.

## Basic mental model

```text
URL changes
  ↓
Router matches route
  ↓
Correct component/page renders
```

## Common pieces
- `BrowserRouter`
- `Routes`
- `Route`
- `Link`
- `NavLink`
