# React + Vite Complete Course Pack (v3)

This pack is a **zero-to-solid-foundation + early advanced** study system for learning the React + Vite ecosystem properly.

It is built for someone who:
- is starting from zero
- wants one structured path instead of random tutorials
- wants definitions, mental models, examples, diagrams, quizzes, exercises, debugging notes, and project plans
- wants content first, not flashy UI nonsense

## Brutal truth

If you skip JavaScript and try to “learn React directly,” you will become a copy-paste developer who breaks on basic debugging. That path is trash.

Correct order:
1. web basics
2. JavaScript basics
3. React fundamentals
4. tooling
5. data flow
6. routing/forms/APIs
7. shared state/hooks
8. projects/debugging

## Quick Mental Model

```text
Browser understands:
HTML + CSS + JavaScript

React:
A JavaScript library for building UI

Vite:
A tool that runs and builds frontend projects

Node.js:
A runtime that lets JavaScript run outside the browser

npm:
A package manager used to install packages and run scripts
```

---

## Quick command flow

```bash
npm create vite@latest my-app
cd my-app
npm install
npm run dev
```

- [ ] I understand what HTML, CSS, and JavaScript do

- [ ] I understand JSX, components, props, and state
- [ ] I understand `npm install`, `npm run dev`, and `npm run build`
- [ ] I understand React hooks at beginner level
- [ ] I understand routing, forms, fetch, and localStorage basics
- [ ] I understand Context and custom hooks at beginner-intermediate level
- [ ] I can create and run a React + Vite app locally
